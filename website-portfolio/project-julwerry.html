Интернет магазин Julwerry

Веб-сайт интернет-магазина, предназначенный для просмотра и выбора украшений. Пользователь может просматривать каталог товаров, добавлять их в корзину и оформлять заказ.

Перейти на сайт

Описание проекта

Julwerry — это демонстрационный интернет-магазин ювелирных украшений. Проект был создан для отработки и закрепления навыков разработки базовой логики электронной коммерции.

Сайт позволяет пользователям просматривать информацию о товарах, добавлять их в корзину и оформлять покупку. Основное внимание в проекте уделено структуре страниц интернет-магазина, организации взаимодействия пользователя с товарами и реализации взаимодействия пользователя с корзиной.

Год реализации: 2025

Основные возможности

  • просмотр каталога товаров
  • отображение названия, описания и цены изделий
  • отображение количества товаров в корзине
  • добавление товара в корзину
  • просмотр краткой сводки заказа
  • базовая демонстрация товаров

Процесс разработки

В рамках проекта была разработана интерфейс и структура интернет-магазина, включая все ключевые элементы работы сайта.

Разработка включала:

  • — создание структуры страниц интернет-магазина
  • — разработку каталога товаров (добавление товаров в корзину)
  • — просмотр детальной сводки товара

Основная логика работы сайта была реализована с использованием JavaScript.

Проект размещён на платформе GitHub Pages и доступен для просмотра в браузере.
